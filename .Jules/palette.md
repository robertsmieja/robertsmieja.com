## 2024-05-18 - Replacing native a tags with ExternalLink component
**Learning:** Found several native `<a>` tags for external links in `src/pages/presentations.tsx` and `src/pages/projects.tsx` and `src/components/header.tsx` that don't have visual indicators or comprehensive ARIA labels. The project has an excellent `ExternalLink` component (`src/components/externalLink.tsx`) that already handles `target="_blank"`, `rel="noopener noreferrer"`, visually appends an external link icon, and accepts an `ariaLabel`.
**Action:** Replace all native `<a>` tags pointing to external resources with `<ExternalLink>` to improve visual consistency and accessibility. Always provide an explicitly calculated `ariaLabel` like `External link to [Resource Name] (opens in a new tab)`.
